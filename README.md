🚀 HireForgeAI

AI-powered recruitment tool for automated CV screening and candidate ranking.

🔗 Live Demo: https://syedtwasti.github.io/HireForgeAI/

📌 About

HireForgeAI is a lightweight web-based platform that streamlines early-stage hiring by using AI to analyze and rank resumes against job requirements.

The goal is simple:
cut down manual screening time and improve hiring decisions with data-driven insights.

⚙️ Features
📄 Resume Screening – Analyze CVs against job descriptions
📊 Candidate Ranking – Score and rank applicants automatically
⚡ Fast Processing – Handle multiple CVs in seconds
🎯 Bias Reduction – Standardized evaluation logic
🧠 AI Integration Ready – Built to plug into NLP/ML pipelines
